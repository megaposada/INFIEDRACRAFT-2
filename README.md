# INFORMACION

- Despues de mucho tiempo, el addon de InfiedraCraft 2 porfin sale a la luz, con los 24 dias de dificultad vistos en la serie original del **27/07/24 al 19/08/24**
- El addon es de uso publico, se pueden hacer eventos y series siempre y cuando se respeten los creditos al creador megaposada
- Dentro de un tiempo se actualizara el addon y se le agregara mas contenido para tener la version definitiva.
